# Data

## `DB.xlsx` — Extraction Database

Structured extraction spreadsheet used as input for all synthesis analyses.

### Sheets

| Sheet | Rows | Description |
|---|---|---|
| `DATA_EXTRACTION` | 70 | Included studies × 29 extracted variables (bibliographic metadata, method characterization, empirical context, evaluation practice, findings/limitations) |
| `DOMAIN` | 65 | Application-domain mapping for studies reviewed in §6 (Article, URL, MAIMs used, Domain, Sub-domain) |
| `Others` | 14 | Supplementary records (semi-agnostic and borderline methods excluded from the main corpus) |
